Коммуналка

версия: 1.0

Программа позволяет вести статистику по коммунальным расходам, отслеживать и анализировать динамику расходов по счётчикам.

Сайт разбит на разделы, переход между которыми осуществляется через главное меню. Учет может вестись в разрезе нескольких квартир. Все данные сохраняются с привязкой к текущей (выбранной) квартире.

Перед началом ведения учета необходимо настроить программу, внести все необходимые счетчики и услуги.

Счетчики

В разделе "Счетчики" необходимо добавить все приборы учета, по которым будет вестись учет. В окне создания счетчика можно выбрать его тип, указать название (будет использоваться для отображения в списке), начальные показания (на момент внесения счетчика), признак учета общедомовых нужд. Для счетчиков тепловой энергии можно задать коэффициент перевода единиц измерения, который по умолчанию равен 1. Данный коэффициент позволяет переводить показания счетчика из одних единиц измерения в другие (например, кВт*ч в Гкал). Там же необходимо настроить тариф. Двухставочный и трехставочный тариф используются в тех случаях, когда на потребление ресурса установлена социальная норма. Позднее можно будет отредактировать все поля счетчика, кроме поля "Тип счетчика". После заполнения всех данных необходимо сохранить их. После создания счетчиков они появятся в списке. При нажатии на счетчик отображается область со списком всех показаний данного счетчика с момента его создания, упорядоченными по убыванию даты. Их можно редактировать и удалять, как и сами счетчики.

Услуги

Следующий раздел - "Услуги". Сюда заносятся все услуги, для которых не установлены или не предназначены счетчики. На форме создания услуги имеются четыре способа расчета стоимости услуги: фиксированная стоимость, по количеству жильцов/площади (сюда попадают все услуги, расчет стоимости по которым ведется по формуле "тариф*коэффициент"), водоотведение (необходимо выбрать из перечня список счетчиков, из которых будут браться данные), подогрев воды. Изменить способ расчета в дальнейшем будет нельзя.

Счета

Когда настроен список квартир, внесены все счетчики и услуги, можно переходить к ведению учета. Осуществляется он в разделе "Счета". Счет представляет собой набор показаний счетчиков и список услуг. При создании счета нельзя использовать уже существующие показания счетчиков, нужно добавить новые. Можно пропустить часть счетчиков и/или услуг (нажать на красный крестик). В дальнейшем можно будет добавить пропущенные показания и услуги, открыв счет для редактирования. При нажатии на ранее добавленные показания счетчика, они откроются для редактирования. В процессе редактирования/добавления счета можно изменять тарифы счетчиков и услуг. Если тариф изменяется в режиме добавления нового счета - он применится к текущим показаниям счетчика и сохранится на будущее. Если же тариф изменяется в режиме редактирования счета - он изменится только у данных показаний счетчика/услуги.


На новые показания счетчиков накладывается ряд ограничений. Дата новых показаний должна быть больше даты начальных показаний. Нельзя создать несколько показаний с одинаковой датой. Сами показания должны быть не меньше предыдущих, которые высчитываются по дате и отображаются в нижней части страницы. В поле "Показания" можно добавлять как абсолютное (текущие показания счетчика), так и относительное (разница между текущими показаниями и предыдущими) значение с ведущим знаком "+", например, "+3.65".
